Formulario HTML

Formulários em HTML são seções que coletam informações em campos

Aplicações

Formulários em HTML são comumente utilizados para receber informações do usuário como: Nome, Senha, CPF, etc.

Tags
Form
Form é a tag principal de um formulário e serve como container para as outras tags de entrada de dado.
Input
input possui vários tipos, todos voltados para a inserção de dados
Text
Recebe um texto de linha única
Radio
Marcações de escolha única
Checkbox
Marcações de escolha múltipla
Submit
Botão de envio de dados
Label
Label é uma tag de acessibilidade, ela serve para ser falada em voz alta e para servir como redirecionamento para a caixa de inserção de dados quando for clicada
Text area
Text area é uma tag para fazer uma caixa de texto maior, muito útil para feedbacks, reviews ou descrições, geralmente não obrigatória
Select
Select define a criação de uma lista "drop-down" com opções omitidas até o clique do usuário
Option
Option é uma tag posicionada dentro da "Select" para definir as opções da mesma
Atributos
Action
Action define para onde será mandado os dados do formulário, geralmente para um arquivo no servidor
Method
Method especifica o método HTTP a ser usado para os dados, há dois tipos:

Get - Envia dados em variáveis de URI

Post - Envia os dados no corpo da requisição sem exibir na URL
Funcionamento
A parte de atributos é responsável por conectar o frontend com o backend para armazenamento dos dados via HTTP
