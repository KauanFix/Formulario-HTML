Formulario HTML

Formulários em HTML são seções que coletam informações em campos

Aplicações

Formulários em HTML são comumente utilizados para receber informações do usuário como: Nome, Senha, CPF, etc.

Tags
Form Form é a tag principal de um formulário e serve como container para as outras tags de entrada de dado.
Input input possui vários tipos, todos voltados para a inserção de dados
Text Recebe um texto de linha única
Label Label é uma tag de acessibilidade, ela serve para ser falada em voz alta e para servir como redirecionamento para a caixa de inserção de dados quando for clicada
Text area Text area é uma tag para fazer uma caixa de texto maior, muito útil para feedbacks, reviews ou descrições, geralmente não obrigatória
